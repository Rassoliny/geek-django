Зеркало

Описание товара:

Зеркало используется как зеркало, а еще красивое.

Характеристики товара

  • Размер 50 * 50 см.
  • Материал: дерево + стекло.

Подробное описание товара

Зеркало создано вручную по уникальной технологии из натуральных материалов.
Отлично впишется в интерьер, где уже есть некоторое количество винтажных элементов.
Не рекомендуется использовать в помещениях с высоким уровнем влажности.
