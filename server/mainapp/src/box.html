Ящик для хранения

Описание товара:

Данный ящик подходит для хранения предметов утвари небольших размеров.

Характеристики товара

Размер Материалы Способ изготовления
40 * 20 см Дерево Вручную
  • Размер ящика 40 * 20 см.
  • Материал: дерево.

Подробное описание товара

Ящик создан вручную по уникальной технологии из натуральных материалов.
Может использоваться для хранения предметов утвари небольших размеров.
Отлично впишется в интерьер, где уже есть некоторое количество винтажных элементов.
Не рекомендуется использовать в помещениях с высоким уровнем влажности.
